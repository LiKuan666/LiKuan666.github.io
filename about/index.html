项目经验

网上花卉商城

  • 开发环境：Eclipse / Maven / MySQL / JDK1.8 / Spring / SpringMVC / MyBatis / Bootstrap
  • 我的职责：从开始的环境搭建，到前后台页面展示，再到最后的后端编写，由本人全部独立完成。
  1. 由于没有什么网页设计经验，所以前台页面全都是模仿小米商城。
  2. 采用md5加密对用户的密码进行加密存储到数据库。
  3. 采用SpringMVC拦截器+Cookie实现用户关闭浏览器三十分钟内自动登录。
  4. 用户注册和用户登录加入短信验证码进行验证登录与注册（阿里短信接口）。
  5. 使用Mybatis-generator自动生成entity实体类、mybatis映射xml文件、mapper接口。
  6. 使用Mybatis分页插件PageHelper，实现快速分页。
